# Exercise 6

In this exercise, the Minimax algorithm is implemented for the game of Connect 4. The alpha-beta pruning technique and a depth-limited version of Minimax are also implemented to improve the algorithmâ€™s performance.

## Files

- `exercise_06.md` - contains the text of the problem for this exercise.
- `Connect4/` - this directory contains definition (template) and solution for the task of this exercise.
- `Connect4/template/` - this directory contains the template for the task of this exercise.
- `Connect4/basic_minimax_solution/` - this directory contains the solution for the task of this exercise with the plain minimax algorithm which works for only very small table sizes.
- `Connect4/optimized_minimax_solution/` - this directory contains the optimized version of minimax algorithm using alpha-beta prunning and limited depth of search, that can work on bigger table sizes.
