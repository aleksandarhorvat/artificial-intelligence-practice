# Exercise 1

This exercise introduces machine learning.

First, the conda environment used throughout the exercises is set up. Next, Jupyter notebooks are introduced, including the VS Code extension. After that, NumPy (the library for scientific computing in Python) is covered.

## Files

- `conda/` - this directory contains materials for the conda package manager
    - `conda-cheatsheet.pdf` - conda cheatsheet with frequently used commands
    - `conda_setup.md` - instructions for setting up the conda environment for these exercises along with basic information about the conda package manager
    - `env.yml` - conda environment file with required packages for these exercises
- `jupyter_notebooks/` - this directory contains materials for Jupyter notebooks
    - `notebook.ipynb` - Jupyter notebook with some basic examples of usage
- `numpy/` - this directory contains materials for NumPy
    - `numpy.ipynb` - notebook with NumPy examples
