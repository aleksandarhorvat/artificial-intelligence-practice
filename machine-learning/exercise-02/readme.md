# Exercise 2

This exercise introduces pandas (the library for data analysis in Python and working with tabular data).
This is an introduction to pandas with basic operations on DataFrame and Series. Additional pandas topics are covered in later exercises.

Content for this exercise is based on the [Kaggle Learn tutorial](https://www.kaggle.com/learn/pandas).

## Files
- `pandas/` - this directory contains resources for pandas
    - `tutorial` - notebooks that explain pandas concepts
        - `1-creating-reading-and-writing.ipynb` - notebook that shows how to create, read, and save pandas DataFrame and Series
        - `2-indexing-selecting-assigning.ipynb` - notebook with examples of indexing, selecting, and assigning for DataFrame and Series
      
    - `exercise_solutions` - notebooks with solutions for the exercises
        - `1-exercise-creating-reading-and-writing.ipynb` - notebook with solutions for creating, reading, and saving DataFrame and Series
        - `2-exercise-indexing-selecting-assigning.ipynb` - notebook with solutions for indexing, selecting, and assigning DataFrame and Series
    - `input` - directory that contains the data needed for the tutorial and exercise notebooks
        - `wine_reviews` - directory containing data about wine reviews
            - `winemag-data-130k-v2.csv` - CSV file with wine reviews data
        - `youtube-new` - directory containing data about YouTube videos
            - `CAvideos.csv` - CSV file with YouTube videos data from Canada
            - `GBvideos.csv` - CSV file with YouTube videos data from Great Britain
